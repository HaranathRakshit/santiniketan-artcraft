# Santiniketan Artcraft - শান্তিনিকেতনের শিল্পকর্ম

A modern, responsive e-commerce website showcasing traditional handcrafted products from Santiniketan, West Bengal. This project celebrates the rich heritage of Bengal's artisans and their world-class handcrafts.

## 🌟 Features

### Design & User Experience
- **Modern, Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Bengali Language Support** - Bilingual interface with English and বাংলা options
- **Smooth Animations** - Engaging hover effects and transitions
- **Accessibility Focused** - Semantic HTML and keyboard navigation support

### E-commerce Functionality
- **Product Categories** - Organized display of different craft types
- **Featured Collections** - Highlighted product showcases
- **Best Sellers Section** - Popular products with pricing
- **Add to Cart** - Interactive shopping cart functionality
- **Artisan Profiles** - Detailed information about local craftspeople

### Interactive Elements
- **Mobile Navigation** - Hamburger menu for mobile devices
- **Language Toggle** - Switch between English and Bengali
- **Smooth Scrolling** - Seamless navigation between sections
- **Notification System** - User feedback for actions
- **Lazy Loading** - Optimized image loading for performance

## 🛠️ Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Modern styling with Flexbox and Grid
- **JavaScript (ES6+)** - Interactive functionality
- **Font Awesome** - Icon library
- **Google Fonts** - Poppins font family
- **Unsplash Images** - High-quality product and artisan photos

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software installation required

### Installation
1. Clone or download the project files
2. Open `index.html` in your web browser
3. The website will load with all functionality ready to use

### Local Development
For development purposes, you can use any local server:

```bash
# Using Python (if installed)
python -m http.server 8000

# Using Node.js (if installed)
npx serve .

# Using PHP (if installed)
php -S localhost:8000
```

Then visit `http://localhost:8000` in your browser.

## 🎨 Design System

### Color Palette
- **Primary Green**: `#2c5530` - Represents nature and sustainability
- **Secondary Green**: `#1e3a22` - Darker shade for hover states
- **Background**: `#f8f9fa` - Light gray for sections
- **Text**: `#333` - Dark gray for readability
- **Accent**: `#28a745` - Success green for notifications

### Typography
- **Primary Font**: Poppins (Google Fonts)
- **Bengali Support**: System fonts for Bengali text
- **Font Weights**: 300, 400, 500, 600, 700

### Layout
- **Container Width**: 1200px max-width
- **Grid System**: CSS Grid and Flexbox
- **Spacing**: Consistent padding and margins
- **Border Radius**: 15px for cards, 25px for buttons

## 📱 Responsive Design

The website is fully responsive with breakpoints at:
- **Desktop**: 1200px and above
- **Tablet**: 768px - 1199px
- **Mobile**: Below 768px

### Mobile Features
- Collapsible navigation menu
- Optimized touch targets
- Simplified layouts for small screens
- Reduced font sizes for better readability

## 🔧 Customization

### Adding New Products
1. Locate the product grid in `index.html`
2. Add new product cards following the existing structure
3. Update images and product information
4. Ensure responsive design compatibility

### Modifying Colors
1. Open `styles.css`
2. Find the color variables at the top
3. Update the hex values to match your brand colors

### Adding New Sections
1. Create new HTML sections in `index.html`
2. Add corresponding CSS styles in `styles.css`
3. Include any JavaScript functionality in `script.js`
